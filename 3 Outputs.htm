innerHTML

innerHTML is the text content in elements. This applies to any element that can contain text including paragraphs, headings, blocks, list items, ...

Editing the innerHTML of a paragraph 

            <script>
            function changeText() {
                document.getElementById("innerHTMLParagraph").innerHTML = "<b>Cha</b><i>nge</i><ins>d t</ins><del>ext</del>"
            }
            function undoChangeText() {
                document.getElementById("innerHTMLParagraph").innerHTML = "Original Text"
            }
        </script>
        <p id="innerHTMLParagraph"> Original <span style="font-family:cursive;">Text</span> </p>
        <button onClick="changeText()"> Change Text </button>
        <button onClick="undoChangeText()"> Reset </button>

Original Text

<div id="test">
</div>

document.write()

Using the write funciton, you can write information to a file, but beware when using. If you write to an HTML document that has already been loaded, you will overwrite the page.

                    <p> 1 + 1 = </p>
                    <script>
                        document.write(1 + 1);
                    </script>

1 + 1 =

window.alert()

Using window.alert() you can make a window pop up within the page with a message.
Refresh this page to see the example in action. 

                <script>
                    window.alert('Make pop-up messages with window.alert()');
                </script>

console.log()

Press F12 to view the HTML of a web page. You can get to the console from there.
Using console.log() You can send messages to the console.

                <script>
                    console.log('This is a log. 5+10=' 5+10);
                    console.log(5+10);
                </script>
