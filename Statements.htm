Variables

A variable stores information. You can store information in a variable and recall it later. Delare a variable with the keyword 'var'. 'var' is similar to 'auto' in C++.

<script>
var n1, n2, n3;
n1 = 1;
n2 = 2;
n3 = 3;
document.write(n1 + n2 + n3);
</script>

Good Practices

Using Whitespace

Using spaces to space out code makes it more readable and less of a hassel.

Which of these is more readable?

<script>var a,b,c;a=5;b=10;c=15;document.write(a+b+c);</script>
or
<script>
    var a, b, c;
    a = 5;
    b = 10;
    c = 15;
    document.write(a + b + c); 
</script>
