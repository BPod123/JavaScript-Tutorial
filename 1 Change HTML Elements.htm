Changing The innerHtml of Elements

You can use JavaScript to put commands into a <button> to make it execute a task when clicked. You can make the button edit the text within an html element with a given id. 

Consider the following code and its following implementation:
            <p id="targetParagraph"> This paragraph will be changed when a button is pressed. </p>

            <!-- Button to change text -->
            <button type="button"

            onClick='document.getElementById("targetParagraph").innerHTML = "This paragraph has been edited with JavaScirpt."'>
            
            Change Text

            </button>

            
            <!-- Button to revert text back -->
            
            <button type="button"
            
            onClick='document.getElementById("targetParagraph").innerHTML = "This paragraph will be changed when a button is pressed."'>
            
            Change text back
            
            </button>

This paragraph will be changed when a button is pressed.


'document'

'document' is a reference to the this document.

getElementById("id string")

getElementById("id string") is used to find elements with the id as a parameter.


innerHTML

innerHTML is the content within an element. The innerHTML of a paragraph is the text within. The innerHTML of an ordered list <ol> would be the list items <li> within the list.
Getting the calling 'document.getElementById("id of element").innerHTML' returns a reference to the innerHTML of an element with the passed in id. You can then manipulate the innerHTML however you want.

Change Attributes of HTML Elements

Similar to the innerHTML of an element, to access an attribute of an element, instead of
'document.getElementBy*means of finding element.innerHTML, just do .attribute.
For example: 

            <img id="Myimage" src="images/mountain.jpeg" alt="image">
            <br>
            <button type="button" onClick='document.getElementById("Myimage").src = "images/RonaldReaganMachineGun.jpg"'>
            Awesome Button</button>

element.style.property = value
