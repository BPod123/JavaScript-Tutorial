Variables

Variables are used to store information. Variables are declared with the keyword var.

var x = 10 ;
document.write(x);

The value 10 is stored in x. x is a pointer to 10. So when document.write(x) is called, it is executed as document.write(10);

JavaScript Operators

  • = → Assignment Operator
    • var x = 5;
var y = 10;
x = y;
document.write(x);


  • + → Addition 
    • document.write(5 + 5);

  • - → Subtraction
    • document.write(10-5);

  • * → Multiplication
    • var x = 5 * 5;
document.write(x);

  • / → Division 
    • var x = 5 / 2;
var y = 5 / 2.0;
var z = 5.0 / 2;
document.write(x + '<br>' + y + '<br>' + z);

  • ** → Exponential 
    • document.write( 5 ** 2 + '<br>');
document.write( 10 ** 3 + '<br>');
document.write( 4 ** 0.5 + '<br>');
  • % → Modulus
    • Modulus is the remainder when dividing
    document.write(5 % 2);
document.write(20 % 3);
document.write(88 % 2); // Using x % 2 is a good way to tell if a number is odd.
  • ++ → Increment 
    • var x = 5;
x++; // x = x + 1;
document.write(x);
  • -- → Decrement 
    • var x = 5;
x--; // x = x - 1;
document.write(x);

    Assignment Operators

    There are aditional assignment operators to manipulate variables that have already been asigned.

  • x += y; → x = x + y;
  • x -= y; → x = x - y;
  • x *= y; → x = x * y;
  • x /= y; → x = x / y;
  • x %= y; → x = x % y;
