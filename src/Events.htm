What is an Event?

An event is something that happens. In the context of JavaScript and web pages, an event can be when a button is clicked, a key is pressed, or when an HTML elements changes. You can set an event as an attribute of an HTML element.

Useful Event Types

onchange
When an element is changed
onclick
When an element is clicked
onmouseover
When the mouse hovers over an element
onmouseout
When the mouse moves off of an element
onkeydown
When the user presses a key
onload
When the page is loaded by the browser

Event Examples

<p id='ex1' onmouseout="document.getElementById('ex1').style.color = 'blue'"
onmouseover="document.getElementById('ex1').style.color = 'green'">This element will turn green when the
mouse hovers over it and blue when the mouse moves off of it.</p>

<p id='ex3' onload="document.getElementById('ex1').style.color = 'red'"
onclick="document.getElementById('ex1').style.fontSize = '200%'">Clicking this paragraph will
make the top paragraph 2x its normal size.</p>
<button onclick="reset()">Reset Examples</button>

}
</script>
